Order Cancel Policy

Zaruri baat: Order place karne ke baad sirf 30 minute ke andar cancel ho sakta hai. 30 minute baad cancel nahi hoga.
  • 30 min andar: Order cancel karwa sakte hain — WhatsApp par invoice number bhejein
  • 30 min baad: Order cancel nahi hoga — processing shuru ho jati hai
  • Cancel hone ki soorat mein koi charge nahi — Cash on Delivery hai isliye koi deduction nahi
  • Agar order ship ho gaya to cancel nahi hoga — delivery ke baad return karein

Return Policy — 3 Din

Order deliver hone ke baad sirf 3 din (72 ghante) ke andar return request bhej sakte hain. App ke My Orders section mein "Return Request" button aayega.
  • Wrong size ya color mila ho
  • Product listing se bilkul mukhtalif item mila ho
  • Item bilkul naya ho — tags lagay hon, use na hua ho
Return kaise karein?
My Orders → Delivered order → Return Request button dabayein

Return Nahi Hoga Agar…

  • 3 din guzar gaye delivery ke baad
  • Item use ho gaya, wash ho gaya ya alter ho gaya
  • Tags ya packaging nahi hai
  • Undergarments / innerwear (hygiene reasons)
  • Sale ya discounted items (sirf defective ho to return hoga)

Return Kaise Karein?

  • Step 1: Delivery ke 3 din ke andar WhatsApp par message karein
  • Step 2: Item ki clear photos + invoice number bhejein
  • Step 3: Hamaari team 24 ghante mein review karegi aur approve karegi
  • Step 4: Hamaare diye huye address par item wapis bhejein
  • Step 5: Exchange ya refund 2–3 din mein process ho jayega

Refund

  • Approved refund 2–3 business din mein process hoga
  • EasyPaisa, JazzCash, ya bank transfer se wapis milega
  • Delivery charges wapis nahi honge — sirf product ki price milegi (jab tak hamaari galti na ho)

Return ya cancel karna chahte hain? Abhi WhatsApp karein!

Privacy Policy

Aapki privacy hamare liye zaroori hai

Aakhri update: May 2026 — HUZEE STORE, Pakistan

Yeh Policy Kya Hai?

Jab aap HUZEE STORE use karte hain — order karte hain, account banate hain, ya sirf browse karte hain — tab aap kuch information share karte hain. Yeh policy batati hai ke hum wo information kaise use karte hain aur kaise protect karte hain.

Hum Kya Collect Karte Hain?

  • Account Info: Naam, email address, password (encrypted)
  • Order Info: Delivery address, phone number, city, province
  • Shopping Activity: Cart items, wishlist, order history
  • Device Info: Browser type, IP address (security ke liye)

Yeh Info Hum Kyun Use Karte Hain?

  • Aapke orders process karne aur deliver karne ke liye
  • Order confirmation aur status update emails bhejne ke liye
  • Account security maintain karne ke liye
  • Customer support provide karne ke liye
  • Website behtar banane ke liye (aggregate data, koi personal info nahi)

Kya Hum Aapki Info Share Karte Hain?

Nahi. Hum aapki personal information kisi bhi third party ko sell ya share nahi karte.
  • Delivery ke liye courier company ko sirf naam, address aur phone share hota hai
  • Payment ke liye koi info share nahi hoti — hum sirf COD accept karte hain
  • Koi advertisement network ya data broker ko information nahi jati

Aapka Data Kitna Safe Hai?

  • Passwords hamesha encrypted form mein store hote hain
  • Firebase (Google) ke secure servers pe data store hota hai
  • Website HTTPS ke zariye secure connection use karti hai
  • Sirf authorized admin aapke orders dekh sakta hai

Aapke Rights

  • Aap apna account aur data delete karwa sakte hain — email par request karein
  • Aap apni personal info update kar sakte hain (Profile page se)
  • Marketing emails se unsubscribe kar sakte hain
  • Koi bhi sawaal ho to hamare saath rabta karein

Firebase Security Rules

Reviews save karne ke liye Firebase Console mein yeh rules paste karein:
Firebase Console → Firestore Database → Rules → Edit rules → Publish 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Products — public read, admin write
    match /products/{id} {
      allow read: if true;
      allow write: if request.auth != null
        && request.auth.token.email == 'zeerakbutt435@gmail.com';
    }

    // Categories — public read, admin write
    match /categories/{id} {
      allow read: if true;
      allow write: if request.auth != null
        && request.auth.token.email == 'zeerakbutt435@gmail.com';
    }

    // Reviews — public read, logged-in user write
    match /reviews/{id} {
      allow read: if true;
      allow create: if request.auth != null;
      allow update, delete: if request.auth != null
        && (resource.data.userId == request.auth.uid
          || request.auth.token.email == 'zeerakbutt435@gmail.com');
    }

    // Orders — user apne orders dekh sake, admin sab dekhe
    match /orders/{id} {
      allow create: if request.auth != null;
      allow read, update: if request.auth != null
        && (resource.data.userId == request.auth.uid
          || request.auth.token.email == 'zeerakbutt435@gmail.com');
    }

    // Users — apna profile khud dekhe/edit kare, admin sab
    match /users/{id} {
      allow read, write: if request.auth != null
        && (request.auth.uid == id
          || request.auth.token.email == 'zeerakbutt435@gmail.com');
    }

    // Settings — public read, admin write
    match /settings/{id} {
      allow read: if true;
      allow write: if request.auth != null
        && request.auth.token.email == 'zeerakbutt435@gmail.com';
    }

    // Limited Time Offers — public read, admin write
    match /limitedOffers/{id} {
      allow read: if true;
      allow write: if request.auth != null
        && request.auth.token.email == 'zeerakbutt435@gmail.com';
    }
  }
}
